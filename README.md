# RELEVA Physiotherapy Clinic Website

This is a modern, responsive, and animated static website built for the "RELEVA Physiotherapy Clinic". The project was developed using React and Vite, with a focus on a clean user interface and smooth user experience.



---

## ‚ú® Features

- **Fully Responsive:** Adapts beautifully to all screen sizes, from mobile phones to desktop monitors.
- **Multi-Page Navigation:** A complete site structure with Home, About, Services, and Contact pages, managed by `react-router-dom`.
- **Smooth Animations:** Subtle page transitions and scroll-reveal animations implemented with `framer-motion` to enhance user engagement.
- **Interactive UI:** Features a mobile-friendly hamburger menu and a functional "Book via WhatsApp" button for easy appointment scheduling.
- **Modern Tech Stack:** Built with modern tools like React and Vite for a fast and efficient development experience.

## üöÄ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You need to have [Node.js](https://nodejs.org/en/) and npm installed on your machine.

### Installation

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/codewith-lionel/RELEVA.git
    ```
2.  **Navigate to the project directory:**
    ```sh
    cd RELEVA
    ```
3.  **Install NPM packages:**
    ```sh
    npm install
    ```

### Running the Development Server

To view the project in your browser, run the following command. This will start a local development server.

```sh
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) (or whatever address the terminal shows) to view it in the browser.
